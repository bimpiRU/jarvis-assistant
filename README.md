# Jarvis Assistant 🤖

Голосовой помощник для ПК на Python с минималистичным Tkinter-интерфейсом. Вдохновлён Джарвисом из фильмов про Железного человека.

## Возможности

- 🎤 Распознавание русской речи через Google Speech Recognition
- 🔊 Синтез речи (TTS) через pyttsx3 — работает офлайн
- 🖥️ Графический интерфейс на Tkinter
- ⚡ Базовые команды: время, дата, открытие приложений, поиск в Google, шутки
- 🔌 Легко расширяется новыми модулями

## Установка

```bash
git clone https://github.com/bimpiRU/jarvis-assistant.git
cd jarvis-assistant
pip install -r requirements.txt
```

> **Важно:** на Windows для работы микрофона нужен `PyAudio`. Если pip не может его собрать, скачайте подходящий `.whl` с [Unofficial Windows Binaries](https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio) и установите вручную.

## Запуск

```bash
python main.py
```

## Использование

1. Нажмите и удерживайте кнопку с микрофоном.
2. Скажите команду.
3. Отпустите кнопку — Джарвис ответит.

### Поддерживаемые команды

| Команда | Действие |
|---------|----------|
| Привет / Здравствуй | Приветствие |
| Который час | Назовёт текущее время |
| Какое сегодня число | Назовёт дату |
| Открой YouTube / Google / браузер | Открывает сайт |
| Открой калькулятор / блокнот / проводник | Запускает программу |
| Найди / Поищи / Загугли [запрос] | Ищет в Google |
| Расскажи шутку | Шутка |
| Кто ты | Представляется |
| Пока / Выключись / Стоп | Завершает работу |

## Архитектура

```
jarvis/
├── config.py      # Настройки
├── speech.py      # Распознавание и синтез речи
├── commands.py    # Обработка команд
├── assistant.py   # Главный цикл ассистента
└── ui.py          # Tkinter-интерфейс
```
