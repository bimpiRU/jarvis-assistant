# Jarvis Assistant 🤖

Голосовой помощник для Windows на Python с минималистичным Tkinter-интерфейсом. Вдохновлён Джарвисом из фильмов про Железного человека.

## Возможности

- 🎤 **Активация по слову «Джарвис»** — непрерывное прослушивание микрофона
- 🔒 **Офлайн-распознавание речи** через Vosk (не требует интернета)
- 🌐 **Онлайн-fallback** через Google Speech Recognition, если Vosk не уверен
- 🔊 Синтез речи (TTS) через pyttsx3 — работает офлайн
- 🖥️ Графический интерфейс на Tkinter в стиле Jarvis
- 🤖 **Интеграция с Kimi Code CLI** — голосовые запросы к AI-ассистенту
- ⚡ Базовые и системные команды
- ✅ **Подтверждение опасных действий** (выключение, перезагрузка, завершение процессов и др.)
- 🚀 Автозагрузка вместе с Windows
- 🔌 Легко расширяется новыми модулями

## Установка

```bash
git clone https://github.com/bimpiRU/jarvis-assistant.git
cd jarvis-assistant
pip install -r requirements.txt
python download_model.py
```

> **Важно:** `download_model.py` скачает ~45 МБ офлайн-модели Vosk для русского языка.

## Запуск

```bash
python main.py
```

### Аргументы командной строки

```bash
python main.py --offline      # Только офлайн-распознавание (Vosk)
python main.py --no-wake      # Отключить активацию по слову
```

Также можно запустить через `start.bat`.

## Использование

1. **Скажите «Джарвис»** — ассистент ответит голосом.
2. Скажите команду.
3. Или зажмите кнопку с микрофоном в окне и говорите.

| Команда | Действие |

| Привет | Приветствие |

|---------|----------|
| Открой YouTube / Google / браузер | Открывает сайт |

#### Программы и папки
| Команда | Действие |
|---------|----------|
| Открой калькулятор / блокнот / проводник | Запускает программу |
| Открой рабочий стол / загрузки | Открывает папку |

#### Система
| Команда | Действие |
|---------|----------|
| Громкость 50 / громче / тише / выключи звук | Управление звуком |
| Яркость 70 / ярче / тускнее | Управление яркостью |
| Система / процессы | Информация о ПК |
| Заверши процесс [имя] | Закрывает процесс (⚠️ с подтверждением) |
| Скриншот | Сохраняет скриншот на рабочий стол |
| Заблокируй компьютер | Win + L |
| Выключи / перезагрузи компьютер | ⚠️ Требует подтверждения |
| Спящий режим | ⚠️ Требует подтверждения |
| Очисти корзину | ⚠️ Требует подтверждения |
| Отмени выключение / перезагрузку | Отменяет запланированное действие |

#### Kimi (AI-ассистент)
| Команда | Действие |
|---------|----------|
| Открой кими | Новая вкладка с интерактивной Kimi |
| Спроси кими [вопрос] | Открывает Kimi с готовым запросом |
| Напиши код [для чего] | Запускает Kimi с запросом на код |

| Команда | Действие |
|---------|----------|

| Автозагрузка | Сообщит статус |

#### Завершение
| Команда | Действие |
|---------|----------|
| Пока / выключись / стоп | Закрывает приложение |

## Подтверждение опасных действий

Перед потенциально опасными командами (выключение, перезагрузка, сон, очистка корзины, завершение процессов) Джарвис:
1. Голосом просит подтвердить действие.
2. Показывает диалоговое окно с предупреждением.
3. Ждёт голосового ответа: **«подтверждаю» / «да» / «выполняй»** или **«отмена» / «нет» / «не надо»**.

Также есть кнопки **«Подтвердить»** и **«Отменить»** в окне.

## Автозагрузка

Чтобы Джарвис запускался вместе с Windows:

```bash
python setup_autostart.py
```

Чтобы отключить:

```bash
python setup_autostart.py off
```

Или голосом: **«Включи автозагрузку»** / **«Выключи автозагрузку»**.

## Интеграция с Kimi

Для работы с Kimi должен быть установлен [Kimi Code CLI](https://github.com/moonshotai/kimi-code). Команды вида **«Спроси кими...»** или **«Напиши код...»** откроют новую вкладку Windows Terminal (или PowerShell) с запущенным Kimi и вашим запросом.

## Архитектура

```
jarvis/
├── config.py            # Настройки
├── speech.py            # Распознавание (Vosk + Google) и синтез речи
├── wake_word.py         # Детектор слова «Джарвис»
├── commands.py          # Обработка команд
├── system_control.py    # Управление системой Windows
├── autostart.py         # Автозагрузка Windows
├── kimi_integration.py  # Интеграция с Kimi CLI
├── dangerous_action.py  # Механизм подтверждения опасных действий
├── jarvis_phrases.py    # Киношные фразы и характер
├── assistant.py         # Главный цикл ассистента
└── ui.py                # Tkinter-интерфейс
```
